📌 Escapada503

Plataforma web desarrollada con HTML, CSS, JavaScript, Django y SQLite para promover el turismo en El Salvador. Permite explorar destinos, consultar guías turísticos, reservar paquetes y gestionar usuarios.

🚀 Funcionalidades:
- Información de sitios turísticos y eventos
- Sección “Sobre nosotros” (misión y visión)
- Guías turísticos con contacto y especialidad
- Paquetes turísticos con sistema de reservas
- Blog con reseñas y valoraciones
- Formulario de contacto
- Perfil de usuario editable
- Historial de reservas
- Cambio de contraseña

🛠️ Tecnologías:
HTML
CSS
JavaScript
Django
SQLite

📚 Proyecto académico:
Este proyecto fue desarrollado como parte de un curso universitario de desarrollo web.
